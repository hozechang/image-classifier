.pth → ONNX 转换脚本

如果你的模型是 PyTorch .pth 格式，运行下面的 Python 脚本将其转为 ONNX 格式： 

python convert_to_onnx.py --arch resnet50 --num_classes 1000 --weights model.pth --output model.onnx

支持的架构: resnet18, resnet34, resnet50, resnet101, vgg16, vgg19, mobilenet_v2, mobilenet_v3_small, mobilenet_v3_large, efficientnet_b0, densenet121, alexnet, squeezenet1_0
